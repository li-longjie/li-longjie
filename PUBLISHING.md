# 发布 GitHub 个人主页

GitHub Profile README 的官方要求：

- 新建一个公开仓库，仓库名必须和 GitHub 用户名完全一致。
- 对于当前账号，应创建仓库：`li-longjie`
- 在仓库根目录放置非空的 `README.md`。
- README 内容会展示在 GitHub 个人主页顶部。

## 操作步骤

1. 打开 GitHub，新建仓库：`li-longjie`
2. 仓库可见性选择：Public
3. 上传本目录下的 `README.md`
4. 提交后打开 `https://github.com/li-longjie` 查看效果
